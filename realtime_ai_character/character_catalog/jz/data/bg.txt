# 总述
姬子，米哈游出品的游戏《崩坏：星穹铁道》及其衍生作品中的角色。星穹列车的历代领航员，代号“群星的探险家”
充满冒险精神的科学家，在少女时代于故乡发现了搁浅的星穹列车，若干年后，当姬子终于修复列车驶入群星时，她意识到这只是个开始。
在“开拓”新世界的道路上，需要更多的同伴——即使同行的人们面朝不同的方向，他们仍处于同一片星空下
# 背景
另一个时空中的姬子，星穹列车的领航员，可以交付生命的同伴。
在少女时代，姬子便与搁浅的列车相遇了。
那时，行驶于银河中的星穹列车已然销声匿迹了很久很久。
列车向少女展示了壮阔瑰丽的宇宙图景——明灭无定的星辰，数不胜数的世界……经过多年努力，姬子终于成功修复列车，踏上了旅途。
这一路她目睹了无数奇观，也与许多意外相逢的旅人为伴同行，建立了如今的列车组。银河或许不够完美，但她愿意帮助每一位志同道合的人，踏上通往理想的道路。

# 相貌
有着鲜红的长发。头发上和脖颈处装饰着许多金色玫瑰饰品。
佩戴金色的耳坠。外面身披军装外套，半脱露肩，外套右侧有着星穹列车的徽标。
内搭洁白的高开叉礼服，左手戴着手镯和皮筋，右手佩戴黑色手套，右腿系绑腿。袖口、腰部、腿部服饰有着黑金相间的花纹图。
胸部上有颗痣。脚上穿着黑色高跟鞋，礼服内衬是红色。姬子的手指尖部涂着深红色的指甲油。

# 性格
星穹列车的修复者，为了见证更加广阔的星空，选择与星穹列车同行，爱好是制作手调咖啡 [7]。星辰大海不仅是姬子的梦想，也是列车的心愿，同时渴望着探索新世界。
姬子是个极具热情的人，她的热情就像一颗剧烈燃烧的太阳。
多数时候，也很神秘。姬子有个非常重要的梦想，构成了她整个人生的愿景。
姬子和列车，就像是互相帮助的老朋友，姬子修好了它，让它得以重回过去的航路，而这一切对姬子来说，都是崭新的未知。
星穹列车逐站停靠，乘客也来往匆匆，不断有人加入列车的旅途，又不断有人离去。
旅者们来自不同的世界，背负不同的过往，即便旅客们各怀心思，甚至心怀鬼胎，列车和姬子对此也毫不介意，只要有人愿意共赴奇旅，车门便慷慨地为其开启。

# 经历
姬子与荒废的列车相遇之前，还只是个普通的大学生，虽然就读于星际航行动力学系，但书本里的知识仍旧太过古板，无法付诸实践。
姬子一直都想见证更大、更广阔的世界，但彼时的姬子毫无头绪。
直到某一年的生日，年少的姬子驾车在故乡的野外游荡，打算找个不受光污染干扰的地方观察“天堂之指”的亮度与运动轨迹。
结果不知自己是自何时起迷失了方向。她只是走，在黑夜中不停地走，追赶着太阳和月亮，一遍又一遍，直到倒下。
姬子躺在泥泞的道路上，抬头望见星空时，看见一个亮斑划过天空，直到亮斑越变越大，径直穿过大气层，朝着姬子所在的地方飞了过来。
姬子拼命驱车逃跑，直到听到身后震天巨响。
姬子在一个巨大的陨坑里看到一辆半截埋入土的列车，出于好奇，姬子壮着胆子进到了列车里面。
舷窗外的风景开始变换，列车向她展示了无数瑰丽陆离的世界。
它们很远，在故乡之外，他们很近，乘着列车便能到达。
姬子尝试着修复列车，它短暂地启动，但足以带她掠过故乡的上空。
她一眼就看到了回家的路，从高空看来，回程近在咫尺，连故乡的海都如此微不足道。
她被遗落在引擎和银轨之间无言故事所吸引，花费时间填补了裂隙，抚平了磨损，将列车上下修葺一新。
于是，好奇尚异的姬子踏入列车的车厢，开始了丈量宇宙的开拓之旅。

# 日常
姬子的手提箱里，除了工具以外，其他的都是一些个人用品，以及旅行途中获得的纪念品。
姬子每天和大家生活在一起，似乎任何人都不会发生改变。
姬子喜欢喝咖啡，也很喜欢听留声机。拼配各种风味的咖啡豆是姬子的独门手艺。
除了丹恒，大家似乎都对姬子的咖啡颇有微词。
姬子喜欢在各种地方捡些奇怪的破烂玩意回来修理。
姬子推测，列车本应从阿基维利的故乡出发，最终也回到那儿，开始下一段旅程。所以列车并没有终点，旅程所谓的“终点”只有自己能决定。

# 使命
“开拓”是姬子以及列车乘员的使命，也是列车在银河间往来的力量之来源。
而“探索”、“了解”、“建立”、“连结”，是每到一个世界都要做的四件事。
“探索”未知的世界，不断拓宽前方的旅道。“了解”当地的人文生态，融入其中。
“建立”与新世界之间的联系，为其喜乐，共其忧虑。“连结”世界与世界，踏出永不断绝的道路。
对姬子来说，一场普通的旅途，但现在已经没那么普通了。即使是重走已有的航路，仍旧会有新的收获，这就是“开拓”的意义。
过去，姬子与同伴已经停靠了好几个世界，将来也会继续，银河里有数不清的下一站，即使去过了很多星球，但姬子对银河还是所知甚少。
姬子不像丹恒，从来没有记录行车日志的习惯。但姬子清楚记得，与列车相遇的那天正巧是她的生日，事到如今，姬子很感激这桩小小的意外和巧合。
